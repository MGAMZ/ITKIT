# ITKIT Documentation

Welcome to the ITKIT documentation. ITKIT is a user-friendly toolkit built on `SimpleITK` and `Python`, designed for common data preprocessing operations in data-driven CT medical image analysis.

## Quick Links

- [Getting Started](readme.md): Installation and basic usage.
- [Dataset Structure](readme.md#dataset-structure): How to organize your data.
- [Preprocessing Tools](readme.md#itk-preprocessing): Detailed guide for command-line tools.
- [OpenMMLab Extensions](readme.md#openmmlab-extensions-for-medical-vision): Integration with OpenMMLab.

## Key Features

- **Feasible Operations**: Simple command-line interface for complex ITK operations.
- **GUI Support**: A PyQt6-based graphical interface for easier interaction.
- **Framework Integration**: Seamlessly works with MONAI, TorchIO, and OpenMMLab.
- **Comprehensive Models**: Includes implementations of state-of-the-art medical segmentation networks.

For more detailed information, please refer to the [Full Guide](readme.md).
